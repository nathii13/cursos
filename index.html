Comandos

Git Clone + enlace ssh de GIT

Replicamos el repositorio creado en GIT, ahora en una carpeta local

Git branch + nombre de rama

creamos nuevas ramas dentro del proyecto

Git switch + nombre de rama

nos permite cambiar entre una rama y otra

Git add .

agrega TODOS los archivos a staging

Git status

Nos indica el estado de los archivos: si están en staging, sin cambios, limpios o en repositorio para producciòn

como navegar entre carpetas con GIT

ls

muestra el contenido de la carpeta local actual

cd + nombre carpeta

permite acceder a través de la terminal a la carpeta indicada

cd ..

permite regresar a la carpeta contenedora de la carpeta inicial. Permite devolverse un nivel
